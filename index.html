Surgimento da internet

Voltar ao topo

HTML, criado por Tim Berners-Lee, significa HyperText Markup Language (Linguagem de Marcação de Hipertexto).

Ele é usado para estruturar e formatar conteúdo na web, permitindo a criação de páginas com textos, links, imagens e outros elementos.

Ao visitar uma página na web, você percebe diferentes tamanhos e distribuições de títulos, parágrafos, imagens, vídeos e outros elementos, o que é possível graças ao uso de HTML para estruturar o conteúdo e CSS para definir o estilo e o layout.

Além do HTML, que estrutura o conteúdo, o CSS é usado para estilizar e definir o layout, enquanto o JavaScript adiciona interatividade. Juntas, essas três linguagens formam a base para a criação de websites modernos.

Como funciona o HTML

Voltar ao topo

Um documento HTML (.html ou .htm) é lido pelo navegador, que renderiza seu conteúdo para que o usuário final possa visualizá-lo.

Um site geralmente é composto por várias páginas HTML. Por exemplo, um site com homepage, página de contato e página de produtos terá ao menos três documentos .html, um para cada página.

O código HTML pode ser escrito em qualquer editor de texto, como o Bloco de Notas.

O que são tags

Voltar ao topo

É possível criar um documento .html usando editores de texto como Sublime Text, NotePad++ ou Bloco de Notas, e ele será renderizado pelos navegadores.

O documento .html é composto por uma série de tags.

Digamos que você queria escrever um parágrafo, usamos a tag <p>, escrevemos o parágrafo e finalmente fechamos a tag com </p>: "Meu primeiro parágrafo".

Dessa forma, ao salvar o arquivo com a extensão .html e abri-lo em um navegador, você verá o parágrafo escrito na tela do navegador.
